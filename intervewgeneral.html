If you have 5 different stylesheets, how would you best integrate them into the site?

My understanding of HTML5 is that it is actually much faster to call in multiple small resources than one big one now. I'd search them for repeated/duplicated items, and call them individually.

Can you describe the difference between progressive enhancement and graceful degradation?

They're mostly just which way you design. progressive enhancement is building off of a base in case features are available, and graceful degradation is building will all features, and planning what happens if they disappear.

How would you optimize a website's assets/resources?

This is a pretty open ended question. I prefer strict organization into folders and sub-folders, using images of the minimum acceptable file-size, and I'd probably try to compare CSS if there were more than one file to make sure nothing is duplicated.

How many resources will a browser download from a given domain at a time? What are the exceptions?

Name 3 ways to decrease page load (perceived or actual load time).

Load the js after the HTML, so that the site is visually ready even if the interactivity lags behind. Minimize the use of libraries (don't use jQuery if you don't need to, and if you use Bootstrap, cut out the parts you don't need). For necessary libraries, research and use the most common CDNs (since the user is more likely to already have them downloaded).
